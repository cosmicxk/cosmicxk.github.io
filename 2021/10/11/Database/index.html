Ubuntu

  • 先在官网 MySQL 下载 DEB Bundle 捆绑包，然后解压会有各种安装包。
  • sudo apt-get update
  • 除了测试包其余都装上，需要依赖就安装依赖
    • sudo dpkg -i xxx.deb
  • sudo apt-get install libaio1 libmecab2 或者 sudo apt-get -f install

安装过程会弹出用户名密码，设置下就行

  • 至此 MySQL 已安装完成
    • 启动服务 service mysql start
    • 检查服务状态 systemctl status mysql.service
    • 进入 MySQL mysql -u root -p
    • -u 的用户是 root，-p 要输入密码

还有一种方法是直接命令行下载，不过我没有成功，就不献丑了！

CentOS

下载

  • 这里得选 RedHat Linux 版本，还是一样，继续下载 RPM 捆绑包

  • 查询是否安装 rpm -qa | grep mysql

  • 解压 tar -xvf mysql-8.0.27-1.el8.x86_64.rpm-bundle.tar

    • 这里注意 -xvf

  • 用 rpm -ivh 进行安装

    • 需要啥安装啥，这里就不一一列出了

  • 因为安装包很全面，就不需要在线安装命令了 yum

更改密码

  • 安装好后设置下权限 chown -R mysql:mysql /var/lib/mysql

    • 启动服务 systemctl start mysqld.service
    • 查看状态 systemctl status mysqld.service
    • 查看 mysqladmin 工具版本 mysqladmin --version

  • 注意这里，mysql

    • 与 Ubuntu 安装不一样，没有弹出让你设置密码的界面，所以显示无权限
    • 网上的解决方法真的是参差不齐
    • 停止服务 systemctl stop mysqld.service
    • 先设置配置文件 sudo vim /etc/my.cnf 在最后加上一行 skip-grant-tables （这一行待会之后还要删除）
    • systemctl start mysqld.service
    • 进入 mysql mysql -u root

  • 修改密码

    • use mysql;
    • 想修改简单的密码得先更改安全变量值
    • 查看安全系数 SHOW VARIABLES LIKE 'validate_password%';
    • 设置为最低 set global validate_password.policy=0;、set global validate_password.length=4;
    • 先把 root 密码置空 UPDATE user SET authentication_string="" WHERE user="root";
    • 更新下 flush privileges;
    • ALTER USER 'root'@'localhost' IDENTIFIED BY 'xing';
    • quit
    • 删除之前配置文件 /etc/my.cnf 新添加的一句
    • 重启服务 systemctl restart mysqld.service

远程连接

  • 用的是小海豚 SQLyog
    • 有点小问题，IP 地址的结尾不能是 0
    • 连接的时候首先要关闭防火墙 systemctl stop firewalld.service
    • 然后进入 mysql 修改 user 表 use mysql;、update user set host = '%' where user = 'root';
    • 之后再重启下

MySQL 命令

  • 切换数据库 use mysql;
  • 查询用户表 select User,authentication_string,Host from user;
    • select * from user; 太多了
  • 查看状态 select host,user,plugin from user;
