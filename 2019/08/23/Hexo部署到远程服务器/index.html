服务器配置

安装 Git

安装依赖包

  • 查看git当前版本的版本号

git --version

卸载旧版本的 Git

yum remove git

下载并解压

  • cd选定文件夹位置

cd /usr/local/src

编译安装

  • 打开环境变量配置文件,在文件底部添加以下配置
  • 脚本文档编辑退出方法 (会频繁用到）
  • 使 git 环境变量生效

source /etc/bashrc

git 新用户及配置

法一

  • 这个方法需要密码，以后每次deploy都要输入密码不太方便
  • 增加一行代码
  • 保存退出

chmod 400 /etc/sudoers

密钥配置

  • 本地

ssh-keygen -t rsa

  • 服务器，将本地.ssh文件夹的 id_rsa.pub 文件内容粘贴到authorized_keys，保存后退出

超级权限root

sudo

  • 修改权限

本地测试是否能连接上服务器

ssh -v git@服务器地址

创建网站目录

法二

网上两种方法创建目录不一样，这里只做参考^-^

法二地址

  • 增加两行
  • 修改文件权限

chmod +x /home/git/hexoBlog.git/hooks/post-receive

建立 git 裸库

  • 查看文件列表

ls

使用 git-hooks 同步网站根目录

  • 填入

chmod +x /home/git/blog.git/hooks/post-receive
